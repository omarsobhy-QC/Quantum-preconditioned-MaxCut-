# Hybrid Max-Cut Solver (Rigetti Track A)

This repository provides a reproducible hybrid pipeline for solving the
weighted Max-Cut problem used in Rigetti's Grid Optimization challenge.

## Features

- Deterministic local solver
- Basin-escape search
- Reproducible metrics
- Visualization output
- QAOA warm-start ready
